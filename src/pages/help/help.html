Help Book details

  • Only 'Title' is mandatory(*) to add a book.
  • 'Search #' field will be used to filter/search books using searchbar.

    •      Adding a book

  • To search on 'Goodreads' you can provide 'ISBN', 'Title', 'Author'
  • First try 'Scan ISBN', if it fails enter it manually and 'Search on Goodreads'.
  • If succeed, book details will be populated.
  • If that fails enter either 'Author' or 'Title' and 'Search on Goodreads'.
  • If this succeed you will be sent to 'Search Results' screen
  • If that fails you have to manually enter the book details
  • Recomending to add at least 'Title','Author' and 'Book Image'
  • Save button is on top right.

    •      Change book cover

  • To change the book cover click on the DEFAULT book image

    •      Select book from search results

  • Search results shows matched books by 'Title' or 'Author' you entered
  • Selected book details will be populated on 'Add book' or 'Edit book' screen

    •      My Book list

  • Swipe enables 'DELETE BOOK' or 'BOOK RETURNED'.
  • If you want to search by english along to native language, update the 'Search #' field with relavant 'Title' and 'Author'

    •      Email book list

  • You can mail your 'BOOK LIST' or 'WISH LIST' to others so they don't have to add the same books you manually entered
  • Images you added manually will also mail along with data file(export.json)

    •      Save files to import

  • If you are using iPhone save only the export.json file to iCloud
  • All the images has to save on gallery
  • If you are using android, save all the files to gallery.

    •      Import books

  • Locate export_books.json file you saved.
  • Select books needs to be imported to which list.
  • Cover image will not be display automatically for books that have manually added book covers. So you have to select the downloaded correct book cover manually from photo gallery and save.

    •      Wish list

  • Once you purchased the book you can move it to 'MY BOOK LIST'

    •      Import from csv file

  • If you have your books on a csv file, make sure you follow these steps
  • Add column name for each column as shown in the image
  • Convert .csv file to .json. You can use online service (https://www.csvjson.com/csv2json) for this.
